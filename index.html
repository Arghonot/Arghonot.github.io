Dragonfly

3 years

Dragonfly is an industrial data visualization and manipulation software. This multi-platform tool (Desktop and VR) was developed with Unity 3D by a dozen developers. Developed within Airbus Defense & Space, it was chosen as the group's official VR visualizer and received several awards.

I had the chance to be part of it's development team for three years. There I was tasked to work on features such as modules including measures, gizmos, various TCP/UDP/REST custom APIs or complementary softwares such as a bug reporter. But I was mainly tasked to work with other teams to bring Dragonfly in production in their field of work.

Technologies

  • C# / C / C++ / Java
  • Visual studio / Unity / Blender / Git / Jira / SQLite
  • VR / AR / UDP / TCP / XAML / WPF / SQL / XML / JSON / Shaders / Threading
